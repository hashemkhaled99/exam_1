# EliteHomes

A responsive, single-page real estate landing page built with **HTML5**, **Bootstrap 5**, and **vanilla JavaScript**.

## 🚀 Live Preview

Open `index.html` directly in your browser, or use a local dev server (e.g., VS Code Live Server).

## 🛠 Technologies Used

- HTML5 semantic markup
- CSS3 custom properties and animations
- Bootstrap 5.3.3 (grid, components, utilities)
- Vanilla JavaScript (DOM manipulation, Intersection Observer)
- Font Awesome 6 icons
- Google Fonts (Inter)

## 📁 Project Structure

```
elite-homes/
├── index.html          # Main landing page
├── css/
│   ├── custom.css      # Theme overrides and component styles
│   └── animations.css  # Scroll reveals and creative animations
├── js/
│   ├── main.js         # Navigation, form validation, header behavior
│   └── animations.js   # Animations and scroll effects
├── img/                # Images and assets
└── favicon.png         # Site favicon
```

## ✨ Features

- Fixed navigation with scroll-aware glass effect
- Hero search form with Bootstrap validation
- Featured properties grid
- Services tabs
- Testimonials carousel
- FAQ accordion
- Contact form with success toast
- Scroll progress bar and reveal animations
- Reduced-motion support for accessibility

## 📌 What I Learned

- How to customize Bootstrap components with CSS variables
- Separating theme styles from animation styles
- Using `IntersectionObserver` for scroll-triggered animations
- Keeping code DRY and maintainable
